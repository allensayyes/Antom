# 🏦 Antom BI Analytics Dashboard

## 项目概述

这是一个专为蚂蚁国际(Antom) BI DA岗位面试设计的高质量商业智能分析平台。该平台基于真实数据，提供全面的业务洞察和战略建议，展示候选人在数据分析、商业智能和业务理解方面的专业能力。

## 🎯 项目特色

- **真实数据驱动**: 所有数据均来自公开渠道，确保真实性和可追溯性
- **全面业务覆盖**: 涵盖全球业务、地区市场、支付方式、商户分析等多个维度
- **专业可视化**: 使用Plotly和Streamlit构建交互式仪表板
- **深度业务洞察**: 提供基于数据的战略建议和业务优化方案
- **完整数据标注**: 每个数据点都标注了来源，确保透明度

## 📊 数据来源

### 主要数据源
- **Antom官方数据**: 业务报告、交易数据、商户信息
- **行业研究报告**: 麦肯锡、普华永道、Statista等权威机构
- **公开财务数据**: 上市公司年报、监管机构报告
- **第三方数据**: 支付行业分析报告、市场研究数据

### 数据质量保证
- 所有数据来源均已验证和标注
- 数据更新至2024年Q4
- 确保数据的准确性和时效性
- 遵循数据隐私和合规要求

## 🚀 快速开始

### 环境要求
- Python 3.8+
- 推荐使用虚拟环境

### 安装步骤

1. **克隆项目**
```bash
git clone <repository-url>
cd 蚂蚁国际
```

2. **安装依赖**
```bash
pip install -r requirements.txt
```

3. **运行应用**
```bash
streamlit run app.py
```

4. **访问应用**
打开浏览器访问: http://localhost:8501
